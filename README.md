# 📱 Responsive Website (Task 4)

This project converts a desktop-only webpage into a mobile-friendly responsive layout using CSS media queries.

---

## 🚀 Features

- Responsive layout for mobile devices
- Flexible design using Flexbox
- Media queries for different screen sizes
- Mobile-friendly navigation
- Scalable images and content

---

## 🧠 Concepts Used

- Media Queries
- Responsive Web Design
- Viewport Meta Tag
- Flexbox
- CSS Units (%, rem)

---

## 📱 How It Works

The layout automatically adjusts based on screen size:

- Desktop → Horizontal layout
- Mobile → Vertical layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3

---

## ▶️ How to Run

1. Open `index.html` in browser
2. Use Chrome DevTools (mobile view)
3. Resize screen to see responsiveness

---

## 💡 Learning Outcome

- Understanding responsive design
- Using media queries effectively
- Adapting layouts for different devices
